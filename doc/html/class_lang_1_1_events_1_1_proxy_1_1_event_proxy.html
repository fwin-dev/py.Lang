Detailed Description

Any method called on this class (besides reserved functions declared in this class) will be proxied out to all EventReceivers registered with the class.

This class is also an EventReceiver itself, as it receives events. Therefore, multiple EventProxy's can be tied together; an event proxy can be registered as an EventReceiver with another EventProxy.
